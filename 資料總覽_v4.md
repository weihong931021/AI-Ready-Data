## 二、三大應用情境

### 情境A：應急導航 — 職災與事故處置

**核心推理鏈**
```
工作情境(S001-S007) → R14 → 事故(A001-A005) → R1 → 法規(L)
  → R2 → 企業義務(D) → R4 → 文件(DOC)
  → R3 → 勞工權益(B) → R5 → 文件(DOC)
  → R12 → 刑事責任(CR) → R13 → 負責人連帶
  → R10 → 民事賠償(CV)
  → R7 ← 裁罰案例(OSHA) → R8 → 法規
```

**IT 情境對應**

| S001 | 系統上線長工時 | A005 過勞 |
| S002 | 夜間值班 on-call | A005 過勞 |

### 情境B：精準止損 — 資遣爭議

**核心推理鏈**
```
工作情境(S008) → R14 → 事故(A006 資遣與解僱爭議)
  → R1 → L023/L024/L025
  → R2 → D009 大量解僱通知 / D010 工資提出 / D011 調解出席
  → R3 → B007 解僱通知資訊權 / B008 工資推定 / B009 調解保障
```

**資料支援**
- 精選案例中有 資遣/解僱 相關的 LBR 裁罰 128 筆（R7/R8 已連結）
- 原始 LBR 資料含 752 筆資遣相關案例（供 VectorRAG 補位）

### 情境C：法規風險雷達 — 產業合規評估

**核心推理鏈**
```
產業(IND_08 資訊服務) → R6 → 違規態樣(VP) → R21 → 法規(L036-L039)
工作情境(S001-S003) → R15 → L033/L034/L035
  → R2 → D006 工時記錄義務
  → R3 → B001 傷病給付
```

**資訊服務業數據**
- 災保法裁罰：687 件
- LBR 工資/工時裁罰：2,363 件（§22 工資 2,584、§24 加班費 2,507、§32 延長工時 1,967）
