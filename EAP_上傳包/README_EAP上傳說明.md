## EAP 上傳順序

1. **PDF** → Data Importer → Import Documents（全 15 份）
2. **CSV/01_節點** → 依序 01→17 建立 Data Flow（Category + Unique Key）
3. **CSV/02_關係** → 依序 R1→R24 建立 Data Flow（連接起終點 Category）
4. **確認視覺化** → Explore 畫布拉出事故類型，確認推理鏈連通
5. **設定 Robot Setting** → 貼入 CLAUDE.md 的系統提示詞

詳細步驟見根目錄 [實作流程.md](../實作流程.md)
