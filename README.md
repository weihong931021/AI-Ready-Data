| 情境 | 名稱 | 解決什麼 |
| --- | --- | --- |

## 快速開始（EAP 上傳）

**所有要上傳的內容都在 [EAP_上傳包/](EAP_上傳包/) 資料夾：**

1. `EAP_上傳包/PDF/` → Data Importer → **Import Documents**（15 份）
2. `EAP_上傳包/CSV/01_節點/` → 依 01–17 順序建立 Data Flow
3. `EAP_上傳包/CSV/02_關係/` → 依 R1–R24 順序建立 Data Flow
4. 設定 Robot Setting、跑 7 題 Demo Query 截圖

詳細步驟 → [實作流程.md](實作流程.md)
