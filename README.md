# AI-Ready-Data 專案入口

本專案是 2026 精誠 AI Ready Data 競賽第二題的 Hybrid RAG 提案與 EAP 上傳資料包，目前主軸調整為「資訊服務業勞動合規與職災風險作戰室」。

## 根目錄保留

- `AGENTS.md`：Codex 使用的目前專案記憶與工作重點。
- `claude.md`：歷史操作紀錄，不作為最新文件維護。
- `實作流程.md`：EAP / Gemini Data 平台實作與上傳步驟。
- `資料/`：最新資料來源、整理產物與 EAP 上傳包。
- `資料_舊版備份/`：舊版或已替換資料備份。
- `scripts/`：資料清理、爬蟲、建包與驗證腳本。

## docs 分類

- `docs/資料說明/`：資料總覽、資料準備計畫與版本說明。
- `docs/提案與簡報/`：提案、Demo、競賽策略、PPT 大綱與設計升級文件。
- `docs/參考案例/`：競賽網頁內容與別組/得獎作品觀察。
- `docs/參考資料/`：工作坊講義、提案範例、上一屆作品 PDF。

## 目前資料包重點

- VectorRAG：32 份 PDF，新增資訊服務業過勞與職場不法侵害官方指引。
- GraphRAG：17 份正式節點 CSV、20 份關係 CSV、2,303 條邊。
- 裁罰資料層：16 份補充 CSV，完整清理表 26,513 筆。
- 自製情境包：補上系統上線、on-call、駐點、IDC、通勤公出、不法侵害的情境、證據、流程與 Demo 問答。
- Demo 主軸：系統上線過勞、駐點客戶現場、職場不法侵害/性騷擾與資訊服務業同業裁罰風險。
