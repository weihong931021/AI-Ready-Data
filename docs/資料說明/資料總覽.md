### 關係檔（[`資料/CSV/關係/`](../../資料_舊版備份/CSV/關係/) — 5 份）

| 檔名 | 邊數 | 連接 |
| --- | --- | --- |
| R1_事故適用法規.csv | 22 | 事故 → 法規條文 |
| R2_法規衍生義務.csv | 9 | 法規條文 → 企業義務 |
| R3_法規衍生權益.csv | 12 | 法規條文 → 勞工權益 |
| R4_義務需文件.csv | 11 | 企業義務 → 文件 |
| R5_權益需文件.csv | 17 | 勞工權益 → 文件 |

→ 完整推理路徑：**事故 → R1 → 法規 → R2/R3 → 義務 / 權益 → R4/R5 → 文件**

人資端走 R2 路徑（向下到義務、罰則）；勞工端走 R3 路徑（向下到權益、給付）。
