## 六、自我審視：缺陷與限制

### ✅ 強項

1. **法條覆蓋完整**：23 部官方 PDF，從勞動法到民刑商法
2. **真實案例量大**：26,513 筆裁罰，**全部都是政府公開資料**
3. **B2B 比例 4:1**：4 種雇主端節點 vs 1 種勞工端
4. **GraphRAG 連得緊**：754 節點 / 2,303 邊，0 個懸空指向

### ⚠️ 弱項與緩解

1. **大法 PDF 含雜訊**
   - 民法 169 頁、公司法 81 頁、刑法 59 頁，多數內容跟職災無關
   - **緩解**：VectorRAG 自動 chunk + embedding，相關段落會浮上來；demo 時用具體 query（如「僱用人連帶責任」）即可命中

2. **司法判決樣本太少**
   - 只 4 筆，且部分不直接相關
   - **緩解**：判決個案性太強，**用「重大職災死亡統計報告」官方數據替代**更具普遍性

3. **高違規列管名冊只 1 頁**
   - 疑似網站只公開部分
   - **緩解**：作為「列管制度存在」的佐證，不當主數據

4. **慰撫金行情數字無確切判決出處**
   - 「200-500 萬」是業界經驗值

3_EAP上傳/CSV/節點/  → 13 份逐一建 Data Model
