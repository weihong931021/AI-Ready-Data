> 目的：把職災/勞動合規 RAG 從一般事故查詢，升級成資訊服務業與系統整合業可用的事故處置與合規風險作戰室。主 demo 先聚焦事故、客戶場域責任與不法侵害；長工時/過勞保留為輔助查詢能力。

## Demo 價值

一般搜尋可以找到單一法條或指引；本系統的差異是把資訊服務業事件轉成：

```text
工作情境 → 事故/法規 → 證據缺口 → 企業處置任務 → 角色分工 → 裁罰/治理風險
```

因此 Demo 不只是回答「哪一條法規適用」，而是產出企業應變作戰清單。加班、長工時與過勞可作為平台查詢能力，但簡報主線優先展示責任邊界較清楚的事故與客戶場域案例。
