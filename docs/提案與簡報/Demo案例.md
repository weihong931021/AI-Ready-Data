# Demo 案例（資訊服務業增強版）

> 主軸改為資訊服務業 / 系統整合業的勞動合規與職災風險作戰室。
> Demo 不再平均展示一般職災，而是用「工作情境 → 事故/法規 → 證據缺口 → 處置任務 → 角色分工」凸顯 RAG 系統價值。

---

## 建議主 Demo 排序

初賽簡報不要平均展示所有案例，建議把鏡頭集中在三個最有說服力的 demo：

| 順序 | Demo | 展示重點 |
| --- | --- | --- |
| 1 | 系統上線長工時 / 過勞風險 | 工時、值班、incident、上線紀錄如何支撐過勞認定與企業處置 |
| 2 | 客戶或主管職場不法侵害 / 性騷擾 | 申訴、調查、保護措施、諮商轉介與避免報復任務 |
| 3 | 駐點工程師客戶機房受傷 | 客戶現場、派工、門禁、合約責任邊界與職災認定 |
| 4 | 資訊服務業同業裁罰風險雷達 | IND_08 + 26,513 筆完整裁罰資料層，證明不是只查法條 |
| 5 | 通勤 / 公出途中事故雙面介面 | 同一事故，人資端輸出義務/風險，勞工端輸出給付/文件 |
| 備用 | 工地墜落重大職災 | 高風險極端案例，保留 8 小時通報與刑責展示 |

這樣安排會更接近別組 Gemini Data 簡報的節奏：先證明平台有跑，再用圖譜與 Q&A 截圖展示洞察。

## 主案例 A：系統上線長工時 / 過勞風險

### 人資 / 專案主管端輸入

```text
我是系統整合公司的專案經理。團隊最近兩個月配合客戶核心系統上線，
工程師每週都有夜間值班和 incident 緊急修復。昨天一位工程師心血管疾病住院，
家屬質疑是過勞。我現在要先做什麼？
```

### 預期 AI 回應重點

- 初步辨識為 `S001 系統上線長工時` → `A005 過勞`。
- 引用職業促發腦心血管疾病認定參考指引與異常工作負荷預防指引。
- 列出證據缺口：出勤紀錄、加班申請、值班表、incident ticket、上線計畫、專案會議紀錄、診斷證明。
- 產出任務：啟動職災初判、蒐集發病前 1 日 / 1 週 / 1 月 / 6 月工作負荷、評估災保給付與雇主補償、通知 HR/PM/法務/職安窗口。
- 延伸企業風險：工時紀錄保存、勞基法加班限制、民事賠償、員工集體訴訟與聲譽風險。

### 串到的資料節點

- 工作情境：S001、S002、S003
- 事故：A005
- 法規：L010、L031、L033、L034、L035、L001、L004、L005、L024、L025
- 證據：E001-E005、E009、E013
- 任務：T001-T005、T012
- 角色：ROLE_HR、ROLE_PM、ROLE_EHS、ROLE_LEGAL、ROLE_MANAGER

## 案例 1：通勤事故（最直觀，建議第一個 demo）

### 1A. 人資端

**使用者輸入**：

```
我是公司人資。員工小張今天上午 7:50 騎機車上班途中被酒駕撞傷，
目前住院中。我該做什麼？
```

**預期 AI 回應結構**：

```markdown
## 事故認定
此案為通勤職災，依勞工職業災害保險職業傷病審查準則§17，
勞工於上下班適當時間從日常居住處所往返就業場所之應經途中
發生事故而致之傷害，視為職業傷害。

## 法定義務
- 工資補償（依勞基法§59）：受傷不能工作期間按原領工資給付
- 醫療補償（依勞基法§59）：必要醫療費用
- 失能補償（如後續評估失能）：依失能等級

⚠️ 此案為單人住院但未涉及死亡或 3 人以上罹災，
   非職安法§37 所定重大職災，不適用 8 小時通報義務。

## 應備文件
- 診斷證明書（醫療院所開立）
- 工資清冊（公司提供，計算給付基礎）

## 違規後果
若未依法給付補償，依勞基法§79 可處罰鍰 2 萬元至 100 萬元，
由地方勞工局處分。

## 延伸問題
1. 員工後續若認定失能，公司還需給付什麼？
2. 員工要求公司代為申請勞保給付，流程為何？
3. 若員工是派遣勞工，補償責任如何分擔？
```

**串到的資料節點**：

- 事故節點：A001 通勤事故
- 法規：L003、L004、L005、L001
- 義務：D002 工資補償、D003 醫療補償、D004 失能補償
- 文件：DOC001 診斷證明書、DOC003 工資清冊
- 罰則：P002 未給付職災補償

### 1B. 勞工端（同一事故，切換身分）

**使用者輸入**：

```
我是勞工。今天上午 7:50 騎機車上班途中被酒駕撞傷，現在住院。
我可以申請哪些補助？
```

**預期 AI 回應結構**：

```markdown
## 事故認定
此案為通勤職災（勞工職業災害保險職業傷病審查準則§17）。
您可同時主張兩類保障：
（A）勞保局災保給付（國家保險體系）
（B）公司依勞基法§59 之補償（雇主責任）

## 可請領給付（向勞保局申請）
- 醫療給付：因職災就診之醫療費用
- 傷病給付：自不能工作第 4 日起，按平均日投保薪資發給
   - 前 60 日：100% 平均日投保薪資
   - 第 61 日起：70% 平均日投保薪資
   - 合計最長 2 年
- 失能給付（若評估有失能）：依失能等級 1-15 級給付
- 照護補助（住院需人照護）

## 申請時效
所有給付時效為 5 年（自得請領之日起算）。

## 應備文件
- 診斷證明書（醫療院所開立）
- 職業傷病給付申請書（勞保局範本）
- 上下班事故陳述書（勞保局範本，通勤職災必備）
- 工資清冊（投保單位提供）

## 受理單位
勞動部勞工保險局

## 申訴管道
若公司拒絕給付勞基法§59 補償，可向公司所在地勞工局申請調解
（依勞資爭議處理法）。

## 延伸問題
1. 我請領傷病給付期間的勞保費誰負擔？
2. 如何判斷我是否符合失能等級？
3. 公司若解雇我，我有什麼保障？
```

**串到的資料節點**：

- 事故節點：A001 通勤事故
- 法規：L003、L004、L005
- 權益：B001 傷病給付、B002 失能給付、B005 醫療給付、B006 照護補助
- 文件：DOC001、DOC002、DOC003、DOC007 上下班事故陳述書

→ **核心展示點**：同一事故、同一知識庫，依身分輸出完全不同視角。這是雙面介面最有衝擊力的 demo。

## 案例 2：工地墜落（重大職災，凸顯 8 小時倒數）

### 人資端輸入

```
我是工地主任。剛才 11:30，作業員小李從鷹架 5 樓墜落送醫，
目前在開刀。我現在該做什麼？
```

### 預期 AI 回應重點

- **🚨 重大職業災害**（罹災 1 人以上需住院）
- **⏰ 8 小時通報倒數**：必須於 11:30 + 8h = 19:30 前通報勞動檢查機構（職安署）
- **現場保留**：禁止移動現場物件直到調查完成
- **罰則**：違反通報義務罰 3 萬-30 萬（職安法§43）；違反危害預防義務（未架安全網）可加重至 1 年以下徒刑或 18 萬罰金（職安法§40）
- **三層補償義務**：勞基法§59 + 災保法給付協助 + 可能民事賠償
- **應備文件**：重大職災通報書、診斷證明、工資清冊、現場照片、安全衛生設施設置紀錄

→ Demo 衝擊力：**畫面顯示 8 小時倒數計時器**，凸顯「期限工具」的價值。

## 案例 3：派遣勞工搬運受傷（連帶責任）

### 勞工端輸入

```
我是派遣勞工，被派到 A 工廠上班，昨天搬機台時壓傷腳。
派遣公司跟工廠互相推託，我該找誰要補償？
```

### 預期 AI 回應重點

- 依勞基法§63-1，**派遣事業單位與要派單位連帶負職災補償責任**
- 您可向**任一單位**主張補償，雙方對外連帶負責
- 若該單位拒絕，可向公司所在地勞工局申請調解（勞資爭議處理法§9）
- 同時可向勞保局申請災保傷病給付（5 年時效）
- 應備文件：診斷證明、上下班事故陳述書（如於通勤）、職業傷病給付申請書

→ Demo 重點：展示 **GraphRAG 在「不同雇主類型」下的推理彈性**。

## 案例 4：過勞猝死（職業病 + 重大職災 + 遺屬給付）

### 人資端輸入

```
我們公司工程師連續 6 個月每月加班 100 小時，今早上班途中
心肌梗塞猝死。家屬找上門說要主張過勞。我們該怎麼辦？
```

### 預期 AI 回應重點

- **過勞職業病認定**：符合勞動部「職業促發腦血管及心臟疾病認定參考指引」過負荷標準（發病前 6 個月平均加班 ≥80 小時）
- **重大職業災害**（死亡）：8 小時內通報職安署
- **應提供**：完整出勤紀錄（勞基法§30 應保存 5 年；未保存罰 2-100 萬）
- **死亡補償**：勞基法§59 規定 5 個月平均工資（喪葬費）+ 40 個月平均工資（死亡補償）
- **附加風險**：可能違反勞基法§32 工時上限規定（再罰 2-100 萬）
- **遺屬可申請**：災保法§49 死亡給付（喪葬津貼 5 個月+遺屬年金或一次金）
- **可能民事訴訟**：遺屬另可請求民法侵權行為賠償

### 同時也跑勞工/遺屬端

```
我先生是工程師，連續 6 個月每月加班破百小時，今早心肌梗塞過世。
我們家可以申請什麼？
```

→ AI 應引導到災保法死亡給付 + 勞基法§59 補償 + 民事賠償三軌路徑。

## 案例 5：化學品暴露職業病（退保後仍可申請）

### 勞工端輸入

```
我之前在電鍍工廠工作 4 年，半年前離職。最近皮膚科醫師說我
這個皮膚炎是職業病。我已經離職還可以申請補償嗎？
```

### 預期 AI 回應重點

- **退保後 1 年內仍可申請**（災保法§78）
- 可請領：職業病醫療給付、失能津貼（若達失能等級）、死亡津貼（若死亡）
- **應備文件**：診斷證明書、職業病職歷報告書（勞保局範本，需詳列工作期間接觸物質）
- **時效**：自得請領之日起 5 年
- 受理：勞動部勞工保險局
- **附加保障**：前公司若違反健康檢查義務（職安法§20），仍可被裁罰 3-15 萬元

→ Demo 重點：展示 GraphRAG 處理**時間維度**（離職後權益、認定時點）的推理力。

## 案例 6（v2 新增）：「我會被罰多少」─ 同業真實案例參照

### 6A. 人資端輸入

```
我是營造業中小企業老闆，工地剛被檢查，工人未戴安全帽且
鷹架沒護欄。我會被罰多少？同業最近有誰被罰過？
```

### 預期 AI 回應結構

```markdown
## 違反法條判定
依您描述，可能違反：
- 職業安全衛生法§6 第1項（雇主必要安全衛生設備義務）
- 營造安全衛生設施標準§19 第1項（高度 2m 以上墜落防護義務）
- 罰則：職安法§43 第2項，**3 萬至 30 萬元**

## 同業真實案例（過去 60 天）
近 2 個月勞動部公告同類型違規 85 件（營造安全衛生設施標準§19
+ 職安法§6 條的常見組合）。

實際處分案例參考：
- OSHA_00177：北邑營造（高處墜落防護不足）─ 已公告
- OSHA_00135：某營造（升降設備防護） ─ 已公告

## 立即建議
1. 立即補強鷹架護欄、發放安全帽
2. 若已發生勞工受傷，記得 8 小時內通報重大職災（職安法§37）
3. 勞動檢查機構查到違規，建議主動配合提供改善計畫
```

→ Demo 重點：**用真實案例 OSHA_xxxxx 證明這不是 AI 編造**，可點開連到原始公告字號。

### 6B. 勞工端輸入（同情境換視角）

```
我老闆是營造業，最近被勞檢來抓到工地很多問題。
這對我有什麼影響？我能要求什麼？
```

### 預期 AI 回應重點

- 雇主違反職安法 = 工作環境**真實高風險**，建議：
  1. 立即要求改善（保留 LINE/Email 證據）

- 萬一發生職災：雇主未盡職安義務，民事求償時是**重要過失證據**，可加重賠償
- 引用真實案例：類似情境的勞工後續主張權益的成功路徑

→ 這是 v2 新解鎖的「**真實裁罰參照**」demo：GraphRAG 先用 603 筆圖譜案例快速對映，完整裁罰資料層另有 26,513 筆支撐同業風險統計。
