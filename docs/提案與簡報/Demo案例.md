## 情境A：應急導航

### Demo A-1：駐點工程師客戶機房搬運受傷

**輸入**
```
我是 IT 公司人資。工程師到客戶機房支援維修，搬伺服器時扭傷腰送醫。
客戶說現場由我們公司管，我們說是客戶指揮。這算職災嗎？誰要負責？
```

**預期 AI 串接節點**
- 工作情境：S004（駐點機房搬運）→ A003（搬運受傷）
- 法規：L001 勞基§59、L002 勞基§63-1 派遣連帶、L008 職安§6、L011 民法§188
- 義務：D002 工資補償、D003 醫療補償、D007 健康檢查
- 文件：DOC001 診斷證明、DOC003 工資清冊
- 刑責：CR002 業務過失傷害
- 任務：T001 職災初判、T004 事故紀錄、T005 法務釐清、T006 PM 調查

**回應重點**
- 先判斷指揮從屬：作業指示由誰發出、現場管控誰負責
- 勞基§63-1：承攬人雇主與要派單位連帶負補償責任
- 列出：派工單、門禁紀錄、合約範圍、診斷證明
- 不處理的後果：CR002 業務過失傷害（1 年以下有期徒刑）

### Demo A-2：客戶/主管職場不法侵害

**輸入**
```
我是人資。駐點工程師反映客戶窗口長期言語辱罵，還有疑似性騷擾。
專案主管怕影響合約希望先不動。公司有哪些法定義務？不做會怎樣？
```

**預期 AI 串接節點**
- 工作情境：S006（客戶/主管不法侵害）→ A005（職場不法侵害）
- 法規：L021 性平§13、L022 性平§38-1、L032 職安設施§324-3
- 義務：D007 健康檢查（含不法侵害管理措施）
- 刑責：CR005 性騷擾防治義務違反
- 罰則：P004（10–50 萬元）

**回應重點**
- 公司不能以合約壓力為由不處理
- 法定義務：受理申訴、組調查小組、保護申訴人、避免報復
- 不處理後果：性平法§38-1 罰鍰 10–50 萬，負責人連帶刑責
- 引用：職安署不法侵害預防指引第四版（VectorRAG）

### Demo A-3：通勤公出事故（雙面介面）

**HR 端輸入**
```
工程師去客戶端開會途中發生車禍，現在住院。
我要怎麼判斷是通勤、因公外出還是職災？公司要做什麼？
```

**勞工端輸入**
```
我是工程師，去客戶端開會途中被撞，現在住院。
我可以申請哪些補助？要準備什麼文件？
```

**核心展示點**：同一知識庫、同一事故，依身分輸出完全不同決策路徑

| | HR 端 | 勞工端 |
| --- | --- | --- |
| 串接節點 | D002–D005 義務 + R22 罰則 | B001–B005 權益 + R24 受理單位 |
| 輸出 | 判斷清單 + 義務 + 文件 + 刑責風險 | 給付類型 + 申請時效 + 文件 + 受理單位 |

## 情境B：精準止損

### Demo B-1：資遣程序合規確認

**輸入**
```
我要資遣一個表現持續不佳的工程師。
公司有哪些程序要走？期限是什麼？不照做會怎樣？
```

**預期 AI 串接節點**
- 工作情境：S008（資遣與解僱爭議）→ A006
- 法規：L023 大量解僱§4、L024 勞動事件§37 工資推定、L025 勞動事件§15 調解前置
- 義務：D009 解僱計畫書（60 日前）、D010 工資出勤資料提出、D011 調解出席
- 權益：B008 工資推定（勞工舉證減輕）、B009 調解程序保障
- 文件：DOC003 工資清冊、DOC010 出勤紀錄

**回應重點**
- 預告期義務（勞基§16）：年資滿 3 年以上需 30 日前預告
- 大量解僱（≥10 人）：需 60 日前書面通知、提出解僱計畫書
- 工資推定（勞動事件§37）：雇主若無法提出工資資料，推定勞工主張成立
- 強制調解前置（勞動事件§15）：起訴前必須先走調解

### Demo B-2：大量解僱流程

**輸入**
```
公司因為某個大專案結束，需要在下個月裁員 15 人。
需要做哪些事？期限是什麼？少做哪一步會被罰？
```

**回應重點**
- 60 日前：書面通知主管機關 + 提出解僱計畫書（L023 大量解僱§4）
- 協商義務：與工會或勞工代表協商（VectorRAG：大量解僱勞工保護法 PDF）
- 工資清冊保存（D010）→ 爭議時舉證
- 未依規定：大量解僱保護法罰鍰 + 可能被認定解僱無效

## 情境C：法規風險雷達

### Demo C-1：IT 公司產業違規風險評估

**輸入**
```
我們是台北市 IT 公司，100 人，主要業務是系統整合和軟體開發，
工程師常有專案趕工加班需求。幫我評估今年最需要注意的勞動法規違規風險。
```

**預期 AI 串接節點**
- 產業：IND_08（資訊服務，687 件災保法裁罰）→ R6 → VP_001/VP_004
- 違規態樣 → R21 → L036–L039（災保法行政義務）
- 工作情境：S001–S003 → R15 → L033/L034/L035（工時三法）→ D006
- 裁罰統計：R8 連結 OSHA/LBR 案例的違反法條分布

**回應重點**
- IT 行業近三年 LBR 裁罰：工資§22（2,584 件）、加班費§24（2,507 件）、工時記錄§30（2,289 件）
- 圖譜推理：IT 情境（長工時）→ 工時法規 → 義務清單（出勤紀錄保存 5 年）
- 建議：先確保 DOC010 出勤紀錄完整，再確認 D006 工時記錄保存
